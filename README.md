## 🌐 الواجهة الأمامية لمشروع المكتبة الإلكترونية

هذه الواجهة الأمامية تم تطويرها باستخدام **HTML, CSS, JavaScript (Vanilla)** وتعمل كمستهلك لواجهات API الخاصة بمشروع ASP.NET Web API. تهدف الواجهة إلى تقديم تجربة استخدام سهلة وجذابة لعرض محتوى المكتبة والتفاعل معه.

---

### 🧩 المميزات

- 🏠 صفحة رئيسية تحتوي على نبذة وروابط موجهة.
- 📚 صفحة الكتب: عرض كل الكتب المتاحة من الـ API بشكل منظم وبصور الغلاف.
- 👤 صفحة المؤلفين: عرض المؤلفين مع صورهم وعدد كتبهم.
- 📖 صفحة تفاصيل الكتاب: عرض معلومات الكتاب، المؤلف، رابط القراءة أو التحميل، وتقييمات المستخدمين.
- ⭐ صفحة التقييمات: عرض تقييمات وتعليقات المستخدمين مع تواريخها.
- ⚙️ صفحات تحكم للإدارة (Admin Panel) لإدارة الكتب، المؤلفين، التصنيفات، والعلاقات.
- 🎨 تصميم متجاوب يدعم اللغة العربية واتجاه RTL.
- 🌙 دعم الوضع الليلي (Dark Mode) لبعض الصفحات.

---

### 🧱 التقنيات والأدوات المستخدمة

- HTML5 و CSS3 للتصميم والتخطيط.
- JavaScript (Vanilla) للتفاعل وجلب البيانات من API.
- 
- Flexbox و CSS Grid لتنظيم المحتوى.
- تخزين المعرفات في روابط URL لاستخدامها بين الصفحات.

---

### 🔗 ربط مع API

يتم جلب البيانات من:
- `GET /api/Book/allBooks` — جميع الكتب.
- `GET /api/Book/by-author/{authorId}` — الكتب الخاصة بمؤلف.
- `GET /api/Book/details/{bookId}` — تفاصيل كتاب.
- `GET /api/Book/reviews/{bookId}` — مراجعات كتاب.
- والمزيد من نقاط النهاية الخاصة بالإدارة.

---

### 📌 ملاحظات

- تم الفصل بين الواجهة والباك إند لسهولة التطوير والصيانة.
- الواجهة تدعم لوحة تحكم للمسؤول، ويمكن من خلالها إدارة المحتوى.
- جميع الطلبات تتم باستخدام Fetch API بدون استخدام مكتبات خارجية.

### 🚀 مستقبل المشروع

- يتم تطوير **نظام خاص للكتب المدفوعة**، يتضمن:
  - عرض كتب حصرية تتطلب الدفع لقراءتها أو تحميلها.
  - دمج **بوابة دفع إلكترونية** لشراء الكتب بطريقة آمنة وسهلة.
- دعم لغات إضافية لواجهة المستخدم.
- تعزيز تجربة المستخدم بإضافة توصيات مخصصة وسجل قراءة.
